Command list

Scripting commands:

$[COLOR] - Colors all the text after this command the specified color, available colors:

  • R - Red
  • W - White
  • G - Adjustable
  • B - Blue
  • L - Light blue
  • Y - Yellow
  • P - Purple
  • BL - Black
  • TOK - Tokiria Green
  • LYN - Lynna Maroon
  • IGA - Igatha Purple
  • OKO - Okoldor Teal
  • RAINBOW - Rainbow
  • GR - Grey
  • PLAYER - Player gold
  • O - Orange
  • HP - Custom Hot Pink

#[COMMAND] - Does various command-related things:

  • WAVE - Wave effect, use XX hex for amplitude
  • SHAKE - Shake effect, use XX hex for strength
  • HEX - use hexadecimal color code ex. #HEX 23EF8A
  • B - Box break, creates a new text box.
  • QE - Quicktext enable.
  • QD - Quicktext disable.
  • PE - Persistent.
  • E - Event.
  • S [XX] - Shifts characters to the right. XX is a hexadecimal number between 00-FF.
  • SFX [XX] - Sound effect command, works in game, not in-tool.
  • I [XX] - Icon tag, displays the icon associated with the icon ID XX in hexadecimal.
  • U - Unskippable.
